# number-converter

Number to text converter

Languages implemented:

- [x] English
- [ ] Lithuanian
- [ ] Polish

## Installation

### npm

```bash
npm install @unknovvn/number-converter
```

### yarn

```bash
yarn add @unknovvn/number-converter
```

### pnpm

```bash
pnpm install @unknovvn/number-converter
```
